# 简易防火墙工具 v0.1

基于UFW的Linux图形化防火墙管理工具，专为Linux Mint设计。

## 功能特性

- **图形化界面**：简洁直观的操作界面
- **防火墙控制**：一键启用/禁用防火墙
- **规则管理**：添加、删除、查看防火墙规则
- **预设配置**：高、中、低三种安全级别预设
- **实时监控**：查看防火墙日志和状态

## 系统要求

- Linux操作系统
- 已安装UFW防火墙
- Python 3.6+
- tkinter库

## 安装方法

1. 确保已安装UFW：
```bash
sudo apt install ufw
```

2. 下载或克隆项目文件到本地

3. 运行程序：
```bash
python3 main.py
```

## 使用说明

### 基本操作

1. **启动防火墙**：点击"启用防火墙"按钮
2. **停止防火墙**：点击"禁用防火墙"按钮
3. **查看状态**：顶部区域显示当前防火墙状态

### 规则管理

1. **添加规则**：
   - 在输入框中输入规则（如：`allow 22/tcp`）
   - 点击"添加"按钮

2. **删除规则**：
   - 在规则列表中选择要删除的规则
   - 点击"删除选中"按钮

3. **常用规则示例**：
   - `allow 22/tcp` - 允许SSH连接
   - `allow 80/tcp` - 允许HTTP连接
   - `allow 443/tcp` - 允许HTTPS连接
   - `deny from 192.168.1.100` - 拒绝特定IP

### 预设配置

- **高安全**：默认拒绝所有入站连接，只允许本地访问
- **中等安全**：开放常用端口（SSH、HTTP、HTTPS）
- **低安全**：允许所有连接

## 命令行示例

```bash

python3 main.py

```

## 注意事项

1. **权限要求**：修改防火墙规则需要管理员权限
2. **网络连接**：错误的规则可能导致无法远程连接
3. **备份规则**：重要系统建议先备份现有防火墙配置
4. **测试环境**：建议先在测试环境中验证规则

## 故障排除

1. **无法启动**：
   - 检查Python版本：`python3 --version`
   - 检查UFW安装：`which ufw`

2. **规则不生效**：
   - 确认防火墙已启用
   - 检查规则语法是否正确
   - 查看防火墙日志了解详细信息

3. **界面异常**：
   - 确认tkinter已安装：`python3 -c "import tkinter"`
   - 重启程序尝试

## 技术架构

- **ufw_manager.py**：UFW命令封装
- **firewall_gui.py**：图形界面实现
- **main.py**：主程序入口
