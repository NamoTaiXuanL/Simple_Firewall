# 简易防火墙工具开发记录

## 项目概述
基于UFW的Linux图形化防火墙管理工具，专为Linux Mint设计，提供类似Windows防火墙的使用体验。

## 完成内容

### 1. 核心模块
- **ufw_manager.py**: UFW命令封装类
  - 防火墙状态管理（启用/禁用）
  - 规则管理（添加/删除/查询）
  - 日志查看功能
  - 预设配置支持

### 2. 图形界面
- **firewall_gui.py**: tkinter图形界面
  - 状态显示区域
  - 快速操作按钮
  - 规则管理界面
  - 日志查看面板
  - 多线程处理避免界面卡顿

### 3. 主程序
- **main.py**: 程序入口
  - 环境检查
  - 异常处理
  - 窗口居中显示

### 4. 文档
- **README.md**: 使用说明文档

## 技术特点
- 小步快跑开发原则
- 最小化功能实现
- 代码简洁易维护
- 多线程界面响应
- 完善的错误处理

## 使用方法
```bash
python3 main.py
```
